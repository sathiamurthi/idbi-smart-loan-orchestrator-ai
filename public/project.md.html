🏆 IDBI Bank Hackathon Project Roadmap

Lending Orchestration Control Panel

"We are not building another loan dashboard. We are building the AI Operating System that enables IDBI Bank to predict, orchestrate, and optimize lending decisions with explainable AI."

Open Live Portal

⚠️ Problem Statement

As loan volumes increase across retail, MSME, and corporate banking, the lending process has become increasingly complex. Loan applications pass through multiple stakeholders—including Relationship Managers, Credit Officers, Risk Analysts, Operations Teams, and Branch Managers—each working on different systems with limited real-time visibility.

Approvals involve manual coordination, delayed agreements, compliance bottlenecks, and inconsistent credit assessments.

🎯 Platform Vision

The IDBI Lending Orchestration Control Panel is more than a dashboard. It is an AI-powered decision intelligence platform that unifies people, processes, and data to make lending faster, smarter, and more transparent. By combining orchestration, predictive analytics, and explainable AI, it empowers every stakeholder—from relationship managers to executives—to make better decisions with confidence.

Transforms fragmented lending checkpoints into a unified, policy-driven multi-agent routing engine.

⚙️ High-Level System Architecture

CUSTOMER PORTAL (App Submit, KYC PDF upload)
        │
        ▼
COORDINATOR AGENT (ORCHESTRATOR) ➜ Reads Policy Threshold Rules
        │
        ├── OCR Agent (Extracts Net Salary, Employer Name)
        ├── Document Validation Agent (Simulates missing files/KYC check)
        ├── Salary Analysis Agent (Calculates monthly net savings index)
        ├── Bank Statement Agent (Classifies active loans from narrations)
        ├── Credit Assessment Agent (Simulates bureau API CIBIL credit score)
        ├── Employer Verification Agent (Checks lists for MNC/PSU/Private)
        ├── Eligibility Agent (Salary income multiplier check)
        ├── Risk Assessment Agent (Deducts points for volatility flags)
        └── Loan Recommendation Agent (Generates final recommended amount & APR)
        │
        ▼
DECISION ENGINE (Auto-Approve / Escalation Review / Auto-Reject)
        │
        ▼
MANAGER DESK (Human-In-The-Loop review, overrides approved limits/rates)
        │
        ▼
JSON DATABASE (browser.localStorage applications & logs tables)
